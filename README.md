# Auto-evoloop

Auto-evoloop is an open-source framework for evaluating AI agents, analyzing failures, planning targeted fixes, and validating improvements through iterative experiments.

This public repository is being initialized as the clean open-source home for the core framework. Internal evaluation data, customer conversations, traces, credentials, proprietary prompts, and private deployment code are intentionally excluded.

## Current Status

This repository is in public bootstrap mode.

- The license and project governance files are public.
- The first source release will be published from a clean history after sanitization review.
- No private repository history will be imported.

## Intended Scope

Auto-evoloop focuses on reusable infrastructure for agent improvement workflows:

- Evaluation runners and scoring adapters.
- Failure analysis workflows.
- Fix planning and execution loops.
- Experiment reporting conventions.

Out of scope for the public repository:

- Real customer or user conversations.
- Business-specific evaluation datasets.
- Runtime secrets, environment files, trace exports, or production endpoints.
- Private agent implementation repositories.
