# S1 Reflection: Evidence Readiness

Use after `auto-trace-prep` produces `analysis_manifest.json`.

## Checks

- Selected case IDs are stable and traceable to input rows.
- Expected behavior is present or explicitly marked missing.
- Trace files are named and cleaned.
- Schema assumptions are documented.
- The manifest states whether human scope confirmation is complete.

## Stop Conditions

- Selected cases are ambiguous.
- Trace files cannot be matched to cases.
- Inputs contain private or real user data that is not approved for the target
  repository.
- The runner command or data source cannot be reproduced.

## Recommendation Format

```markdown
### S1 Reflection
- Recommendation:
- Evidence reviewed:
- Blocking issues:
- Assumptions:
- Required next action:
```
